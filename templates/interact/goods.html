Contribute Goods, Services, & Money

Crypto.is is able to run security and anonymity services based off donations, but primarily its members have chosen to associate services they run personally with the project to promote its growth. If you have spare cash, bandwidth, a server, or the inclination to run a service yourself - please get in touch. We'll be happy to help you set it up, or even administer it for you. The end goal is simple: we want to grow the network of services available.
